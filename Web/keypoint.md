# CSS 中的 em
CSS 中的 em 单位是一个相对的测量单位，用于测量网页上元素的大小，主要是字体大小。因为它是相对于其父元素而言的，所以 1 个 em 等于父元素中设置的字体大小。
这意味着如果你将父 div 中的字体大小设置为 20px，并将子 div 的字体大小设置为 2em，那么子 div 中的字体大小将等于 40px。
